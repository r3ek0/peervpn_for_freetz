peervpn_for_freetz
==================

Integrate peervpn into Freetz for Fritzbox

Installation:
-------------
1. Install a Freetz build environment
2. clone this repository
3. cd into clone-dir
4. copy the peervpn directory to <path_to_freetz_clone>/make/
5. cd into <path_to_freetz_clone>
6. make menuconfig
7. Select "Packages"->Peervpn
8. Select other cool stuff
9. Save and exit
10. make
11. flash firmware to your fritzbox

More info on how to build a Freetz image for your FritzBox is here:
http://freetz.org/wiki/help/howtos/common/newbie.en

A ready-to-use virtualbox image with a working build environment can be found here:
http://sourceforge.net/projects/freetz-linux/
